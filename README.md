# AI Market Insights

服装品类市场洞察 + 商品结构规划 + 选款测款自动化原型。

## 项目简介

这是一个单页 HTML 数据产品原型，用于模拟服装品牌内部的市场洞察工作流。页面围绕三个核心场景设计：

1. **市场洞察**：从品牌、价格带、款式、功能、面料、填充物、设计、颜色等维度观察市场变化。
2. **商品规划**：基于历史销售、本年销售表现和未来目标，规划老品保留、新品开发和品类结构。
3. **选款测款**：录入新品属性后，模拟评估市场前景、价格适配、品牌适配、风险点和测款建议。

## 当前功能

- 深色左侧三级导航
- 顶部全局筛选器
- 市场洞察八维度表格
- 价格带近两年柱状对比图
- 点击维度行跳转到维度详情
- 商品规划模拟表格
- 新品录入与评分卡
- 测款建议、风险提示、竞品参考
- OpenAI API 预留函数 `analyzeWithOpenAI(rows, taskType)`

## 数据说明

当前版本使用前端 JS 模拟数据，主要数据结构包括：

- `marketDimensionData`
- `planningData`
- `newProductEvaluationData`

后续可以替换为 Excel 上传、数据库接口或 OpenAI API 解析后的结构化 JSON。

## 使用方式

直接打开：

```text
market-trend-dashboard.html
```

即可在浏览器中查看原型。
